Processes a frame on the passed arena. 

This function is called once every frame on the main controller to process one frame on the arena. First checks collisions with the enemies, reducing the health of the player with a cooldown of one second. Also handles the collisions with the heart power-\/ups that recover the player\textquotesingle{}s missing health. ~\newline
 It then handles the enemy and heart spawning by calling the functions \doxylink{enemy__controller_8c_a60b46083beb9a9caca98687b50a23847}{handle\+\_\+enemy\+\_\+spawning()} and \doxylink{arena__controller_8c_aa5a4d603090c9ce2c25707be8bdaf9e5}{handle\+\_\+heart\+\_\+spawning()}, while also calling the \doxylink{arena__controller_8h_a63dac5d250ad059af842c4b6292260f1}{handle\+\_\+auto\+\_\+attack()} function to execute the normal attack on cooldown. ~\newline
 At the end, it moves all the entities associated with the arena by calling \doxylink{arena__controller_8c_a2fcd7bed1ee21149f594ab5663588ec2}{arena\+\_\+move\+\_\+entities()}.

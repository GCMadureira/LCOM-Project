◆ handle_enemy_collisions()

bool handle_enemy_collisions ( arena * arena)

Checks the collisions of the active enemies with the player and with attacks.

Iterates through the list of active enemies and decreases their health if hit by an attack, using check_attack_collison() to check, removing them from the arena if the health reaches 0.
If the enemy was not removed, calls check_player_collision() to check if the it collided with the player.

Parameters
arenaA pointer to the arena where the collisions will be checked
Returns
true if an enemy collided with the player, false otherwise

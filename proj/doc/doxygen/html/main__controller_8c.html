Detailed Description

Source code to control the program.

This file handles the logic for the whole program, controlling the menus and the arena by calling the functions in arena_controller.c. It also calls the drawing functions.

◆ process_frame()

int process_frame ( )

Processes a game frame.

If the game_state is on a menu, calls process_menu(), else calls process_game().
After processing the input events, it calls arena_process_frame() if the game_state is on the arena and calls game_over_menu() if the player reaches 0 health.
At the end, draws the menu or arena, depending on which is active.

Returns
0 in case of success, 1 otherwise
